# QA API Tests Portfolio

Projeto de automação de testes focado em validação de APIs utilizando Cypress e Postman.

## Tecnologias
- Cypress
- Postman
- JavaScript

## O que foi testado
- Endpoints de usuários e posts
- Validação de status code
- Validação de estrutura de resposta

## Como executar

```bash
npm install
npx cypress open
